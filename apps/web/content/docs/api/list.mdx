`files.list(options?)`

Cursor-paginated listing with prefix filter. Each item is a [`StoredFile`](/api#the-storedfile-type) with a lazy body accessor.

```ts lineNumbers
const { items, cursor } = await files.list({
  prefix: "avatars/",
  limit: 100,
});

if (cursor) {
  const next = await files.list({ prefix: "avatars/", cursor });
}
```
